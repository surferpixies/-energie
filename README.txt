ÉNERGIE & REPAS — V1.1

Nouveautés :
- interface modernisée et navigation par onglets;
- mode clair, sombre ou système;
- modification et suppression des repas;
- recherche dans l'historique;
- statistiques et graphique des 14 derniers jours;
- associations par aliments récurrents;
- export/import JSON et export CSV;
- conservation des données de la V1 grâce à la même clé de stockage.

Pour mettre à jour GitHub :
1. Téléverser les fichiers de ce dossier à la racine du dépôt.
2. Remplacer les fichiers portant le même nom.
3. Commit changes.
4. Attendre 1 à 3 minutes.
5. Sur iPhone, fermer et rouvrir l'app. Si l'ancienne version demeure, ouvrir le site dans Safari et actualiser.

Important : les données restent stockées localement sur l'appareil. Faire régulièrement une sauvegarde JSON.
