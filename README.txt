ÉNERGIE & REPAS — V1

Contenu
- Application web mobile installable
- Enregistrement d'un repas avec photo, texte, taille, fatigue avant et sommeil
- Suivi de fatigue à 1 h et 3 h
- Tableau de bord local
- Tendances simples par mots récurrents
- Export CSV
- Fonctionnement hors ligne après la première ouverture

Important
Les données sont conservées localement dans le navigateur de l'appareil. Exportez régulièrement le CSV comme sauvegarde.

Pour l'utiliser sur iPhone
1. Hébergez le contenu du dossier sur un service statique HTTPS (GitHub Pages, Netlify, Cloudflare Pages, etc.).
2. Ouvrez l'adresse dans Safari.
3. Partager > Ajouter à l'écran d'accueil.

La reconnaissance automatique des aliments sur photo n'est pas encore incluse dans cette V1; elle demande un service d'IA côté serveur.
